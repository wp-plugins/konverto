== Description ==

This Plugin to convert URLs from www.joerg-burbach.de to a clickable link to http://www.joerg-burbach.de.

* please activate in Preferences before use *

Optional:
* Set target to new Page
* Add title (which will use the original URL)
* convert mail-addresses to clickable links using mailto

== Installation ==

Upload and activate the plugin. Go to preferences to configure.
